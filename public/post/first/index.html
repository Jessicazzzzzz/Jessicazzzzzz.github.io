hugo搭建博客执行步骤#

下载 hugo

  1. brew install hugo 创建站点
  2. hugo new site myblog
  3. cd myblog
  4. git init
  5. 找到自己想要的主题,根据步骤安装 创建博客
  6. hugo new post/first.md
  7. hugo server -D

部署到gitPage#

先在github 上创建一个仓库,名字必须是username.github.io 这种方式只是将static文件直接上传到github上，而不是将整个文件夹上传,所以不用gitaction 来部署

  1. cd myblog/public
  2. git init
  3. git add .
  4. git commit -m “first commit”
  5. git remote add origin https://github.com/useraname/username.github.io.git
  6. git push -u origin main
